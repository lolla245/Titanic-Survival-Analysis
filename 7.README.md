 # 🚢 Titanic Survival Prediction – Machine Learning Project

## 📌 Overview

This project builds a **classification model** to predict passenger survival in the Titanic disaster using machine learning techniques.
The focus is on building a complete **end-to-end ML pipeline**, including data preprocessing, exploratory analysis, feature engineering, and model evaluation.

---
## 🎯 Objective
The objective is to predict whether a passenger survived the Titanic disaster based on features such as:
* Passenger class
* Gender
* Age
* Family size
* Fare
* Embarked location
---

## 📊 Dataset Information
The dataset contains passenger information from the Titanic disaster:
| Feature     | Description                          |
| ----------- | ------------------------------------ |
| PassengerId | Unique identifier for each passenger |
| Survived    | Target variable (0 = No, 1 = Yes)    |
| Pclass      | Ticket class (1st, 2nd, 3rd)         |
| Name        | Passenger name                       |
| Sex         | Gender                               |
| Age         | Age of the passenger                 |
| SibSp       | Number of siblings/spouses aboard    |
| Parch       | Number of parents/children aboard    |
| Ticket      | Ticket number                        |
| Fare        | Ticket fare paid                     |
| Cabin       | Cabin number (if available)          |
| Embarked    | Port of embarkation                  |
---

## 🧹 Data Preprocessing

* Handled missing values:

  * `Age` → Median imputation
  * `Embarked` → Mode imputation
  * `Cabin` → Dropped due to high missing rate
* Converted categorical variables into numerical format
* Removed irrelevant features (e.g., Name, Ticket)

---

## 📊 Exploratory Data Analysis (EDA)

Key insights from exploratory data analysis (EDA):

* Female passengers had significantly higher survival rates
* 1st class passengers had higher survival probability
* Younger passengers had better survival chances
* Strong correlation between `Pclass`, `Sex`, and survival

---

## 🧠 Feature Engineering

* Encoded Sex using binary encoding (0/1)
* Created meaningful feature groups
* Handled skewed distributions in Fare
* Removed redundant features for better model performance

---

## 🤖 Machine Learning Models

The following machine learning models were implemented and compared:

* Logistic Regression (baseline model)
* Decision Tree Classifier
* Random Forest Classifier

---

## 📈 Model Evaluation

Model performance was evaluated using the following metrics:

* Accuracy Score
* Confusion Matrix
* Precision / Recall / F1-score

👉 Best performing model: **Random Forest Classifier**

---

## 🏆 Key Learnings

* Importance of handling missing data correctly
* Feature selection significantly impacts performance
* Ensemble models outperform simple linear models
* EDA plays a critical role in understanding data patterns
